Ang Sistemang Solar

Sa among Sistemang Solar dunay walo ka mga planeta: Ang Merkuryo, Ang Kabugason, Ang Kalibotan, Ang Marte, Ang Hupiter, Ang Saturno, Ang Urano, ug ang Neptuno, ug usa ka bituon: ang adlaw.

Ang Merkuryo usa ka planeta sa atong Sistemang Solar. Ang Merkuryo ang una nga planeta. Ang Merkuryo ug ang Kabugason duha ka mga planeta. Ang Kabugason ang ikaduha nga planeta. Ang Merkuryo, ang Kabugason, ug ang Kalibotan tulo ka mga planeta. Ang Kalibotan ang ikatulo nga planeta.

Ang Marte ang ikaupat nga planeta. Ang Hupiter ang ikalima nga planeta. Ang Saturno ang ikaunom nga planeta. Ang Urano ang ikapito nga planeta. Ang Neptuno ang ikawalo nga planeta.

Dunay sa atong Sistemang Solar walo ka mga planeta ug usa ka bituon.

Ang ikapito nga planeta, ang Urano, di ba? Oo, ang ikapito nga planeta, ang Urano. Ang ikaunom nga planeta ba, ang Urano? Dili ang ikaunom nga planeta, ang Urano, apan ang ikapito nga planeta.
